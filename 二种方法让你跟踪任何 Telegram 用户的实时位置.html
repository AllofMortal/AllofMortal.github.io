您是否知道 Telegram 不仅仅是一个聊天应用程序？它有一个很酷的功能，称为“地理位置跟踪”。这是一种奇特的说法，表示您可以在地图上找到您的朋友所在的位置。让我们谈谈如何使用这个有趣的功能以及如何保守您自己的位置秘密。Telegram 有一个特殊的技巧可以告诉你谁在附近。这就像在手机上玩捉迷藏游戏一样。您甚至可以假装您的朋友在世界不同的地方。但有一条规则——每 5 分钟只能执行一次。问题是：如果你的朋友不使用 Telegram 或关闭了他们的位置，它就不起作用。

想象一下你正在玩一个游戏，它变得有点太棘手了。这就是 Telegram 最近发生的事情。他们有一个叫做“附近的人”的功能，但是它太强大了，就像超级英雄的能力一样。一年多后，他们意识到了这一点，并削弱了它的威力。现在，当你使用它时，你只能看到某人是否在500米、1公里、2公里之外，等等。这就像看一张更大的地图而不是一张小地图。

他们花了很长时间才意识到自己的错误，但现在你将看不到你的朋友到底在哪里。我不太确定 Telegram 和保密事宜。也可能是俄罗斯和乌克兰媒体突然爆发的巨大负面情绪终于让他们醒悟了。   

   

方法一：使用 Telegram 附近地图

Telegram 附近地图使用 OpenStreetMap 和官方 Telegram 库来查找附近用户的位置。

请注意：Telegram 的 API 不久前进行了更新，使附近的用户距离不太精确，从而妨碍了精确的位置计算。因此，Telegram 附近地图会显示附近的用户，但不会显示他们的确切位置。

它是如何工作的？

每 25 秒，所有附近的用户都会收到来自Telegram 的TDLib。这包括附近每个用户到“我的”位置的距离。通过距三个不同点的三个距离，可以计算附近用户的位置。   

这只会找到已激活附近功能的 Telegram 用户。默认情况下它是禁用的。

1.首先在此处为您的 Telegram 帐户创建 API 密钥。您还需要在您的帐户中添加个人资料图片，并在您的Privacy and Security设置中为每个人启用个人资料图片。   

2. 启动 Geogramint

3. 在设置中，写入您的信息（api_id、api_hash 和电话号码），就像我们在本博客中对方法 1 所做的那样，报告首选项，然后save
